# !Doctype

`<!DOCTYPE html>` is required at the top of all documents --> sole purpose is to prevent browser from switching into so-call "quirks mode"

# `<html>`

`<html>` --> the root of an HTML document

# `<head>`

`<head>` element contains machine-readable information (metadata): title, scripts, and style sheets. There can be only one `<head>` element in an HTML document.\
Elements that can be used inside the `<head>`:

- `<title>`
- `<base>`
- `<link>`
- `<style>`
- `<meta>`
- `<script>`
- `<noscript>`
- `<template>`

# `<body>`

The `<body>` HTML element represents the **content** of an HTML document. There can be only one `<body>` element in a document.
